# VidFast

Video streaming application with TMDB integration.

## Deploy

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/template/Drownek/vidfast?referralCode=)

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/Drownek/vidfast)

## Environment Variables

- `TMDB_API_KEY` - Your TMDB API key

## Local Development

```bash
npm install
npm run dev
```

Server runs on http://localhost:3000
